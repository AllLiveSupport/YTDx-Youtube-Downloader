# Project Brief

## Overview
Best YTDx Pro is a modern, user-friendly desktop application for downloading videos and music from YouTube. Built with Python and PyQt6, it offers a robust interface for high-quality media extraction, supporting playlists, metadata tagging, and multiple formats.

## Core Goals
- **Ease of Use**: Provide a simple, intuitive GUI for complex downloading tasks.
- **Quality**: Prioritize high-quality video (VP9/AV1) and audio extraction.
- **Completeness**: Support playlists, cover art, and metadata (ID3 tags).
- **Reliability**: Robust error handling, auto-retries, and FFmpeg integration for format conversion.

## Key Features
- **Video Download**: Supports resolutions up to 4K/8K (depending on availability), converting to MP4.
- **Audio Download**: Extracts high-quality audio, converts to MP3 or M4A, and embeds cover art/metadata.
- **Playlist Support**: Batch processing for both video and audio playlists.
- **Customization**: Dark/Light themes, multi-language support (TR, EN, ES, RU).
- **FFmpeg Management**: Auto-detection and manual selection of FFmpeg binaries.
